# React Admin Dashboard

This project was created to improve skills in creating React UI and learning MUI library. I used many new libraries to display effective charts, tables ...

## How to Install and Run the Project

You have to clone this repository and run command: npm install  and then npm run start to start web server 

Working project you can find here: https://portfolio-2.witekpruchnicki.pl/ 

## How to Use the Project
![alt text](https://user-images.githubusercontent.com/62050443/210263925-074bf25f-6475-4b43-97ab-727d1615cdd8.png)

You can check tabs on the sidebar menu, they are filled with data. Additionally you can switch between dark mode in the top right hand corner
